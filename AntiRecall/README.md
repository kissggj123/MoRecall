# AntiRecall（QQ防撤回）

这款软件用于Win PC平台，针对QQ撤回功能，模块加载成功后方能使用

## 新功能

- 新增内存补丁模式，防撤回更加高效稳定
- 重写了ui
- 新增恢复QQ自启动功能
